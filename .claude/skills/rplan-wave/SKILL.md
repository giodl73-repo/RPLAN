# RPLAN Wave

Use this skill to create or update RPLAN waves.

## Wave requirements

- Define a product-neutral plan package goal.
- Identify affected crates and consumer repos.
- Break the wave into small pulses.
- Include validation expectations for every pulse.
- Record non-goals so RPLAN does not absorb BISECT application logic.
